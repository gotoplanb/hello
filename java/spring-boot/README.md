# Hello {Name} API with Spring Boot

## Build

1. `make build`

## Run

### Local

1. `make run`
2. Browse to `/greeting` route of [localhost](http://localhost:8080/greeting)
3. Browse to `/greeting?name` route of [localhost](http://localhost:8080/greeting?name=Spring%20Boot)
